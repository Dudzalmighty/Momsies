===========================================
HOW TO USE
===========================================

1. Save all files in the same folder.
2. Open any .html file in your web browser.
3. All pages link to each other correctly.
4. The website is fully responsive (works on mobile).

===========================================
ADDING YOUR OWN IMAGES
===========================================

Create an "images" folder inside the website folder.
Then replace the placeholder URLs with your images:

Current Placeholder → Replace With
------------------------------------------------
placehold.co/600x500/... → images/hero.jpg
placehold.co/400x300/... → images/product-name.jpg
